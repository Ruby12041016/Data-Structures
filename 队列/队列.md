### 队列（Queue）

队列是一种遵循 **先进先出（FIFO, First In First Out）** 原则的数据结构。即最先入队的元素最先出队。

#### 1. 队列的基本操作

队列有以下几种常见操作：

| 操作          | 描述                 |
| ----------- | ------------------ |
| `enqueue()` | 向队列中插入一个元素，通常发生在队尾 |
| `dequeue()` | 从队列中移除一个元素，通常发生在队头 |
| `front()`   | 获取队列头部元素的值，但不删除它   |
| `rear()`    | 获取队列尾部元素的值，但不删除它   |
| `isEmpty()` | 判断队列是否为空           |
| `size()`    | 获取队列中的元素个数         |
| `clear()`   | 清空队列               |

#### 2. 队列的实现方式

* **数组实现队列**：使用数组存储队列元素，队头指针和队尾指针来标记队列的头尾。
* **链表实现队列**：通过链表的头尾指针来实现队列，插入和删除操作时间复杂度为 O(1)。

#### 3. 队列的变种

* **循环队列（Circular Queue）**：避免了在数组实现队列时空闲空间浪费的问题，尾部元素移到数组前端，形成一个环形队列。
* **双端队列（Deque, 双向队列）**：允许在队头和队尾进行插入和删除操作。可以实现为输入输出都可以进行的队列。
* **优先队列（Priority Queue）**：根据元素的优先级进行排序，出队时按优先级顺序。



#### 4. 队列的时间复杂度

* `enqueue()`, `dequeue()`, `front()`, `rear()` 等操作的时间复杂度通常是 O(1)。
* `isEmpty()` 和 `size()` 操作的时间复杂度也是 O(1)。
